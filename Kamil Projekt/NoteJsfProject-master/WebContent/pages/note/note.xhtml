About Note

Note is an application which allows you to assign tasks to other user. But it is not the only one purpose of Note. You are also able to add notes for yourself to remember about most important things in daily life. Easy filtering is provided to quickly navigate in your tasks.

Message from the author

Note was created only as "final paper". Most important thing to learn in development process was how backend part of JSF technology works. I was never huge fan of front-end technologies so it's for sure the weakest part.
